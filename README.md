# Dayton Data Challenge Website

This repository contains the source code and content for the official Dayton Data Challenge website.

The website is built using **Quarto** and hosted through **GitHub Pages**.

## About the Competition

The Dayton Data Challenge is a week-long undergraduate data competition hosted by the Department of Mathematics at the University of Dayton.

Students work in teams to analyze real-world datasets, develop meaningful research questions, communicate insights, and present their findings to judges.

The competition emphasizes:

* Curiosity and exploration
* Data-informed decision making
* Effective communication
* Collaboration and teamwork
* Real-world problem solving

## Website Structure

The website currently includes:

* Home
* About
* Competition
* Timeline
* Judging & Awards
* FAQ
* Contact

## Technology

This site is built with:

* Quarto
* HTML/CSS
* GitHub Pages

## Local Development

To preview the website locally:

```bash
quarto preview
```

To render the website:

```bash
quarto render
```

The rendered site will be generated in the `_site` directory.
